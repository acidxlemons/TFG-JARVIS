Modelos Locales

Uso eficiente de VRAM mediante Q8 y Q4 Quantization con herramientas como Ollama y LiteLLM. (Llama 3.1 & Qwen).

Vector Store

Indexado e inferencia semántica optimizada usando Qdrant con modelos SentenceTransformer (MiniLM).

Observabilidad Transparente

Interfaces gráficas nativas con pgAdmin para trazar el historial relacional conversacional y Qdrant Dashboard para auditar visualmente el subespacio latente vectorial.

Conexiones Multicanal

Integración automática (Webhooks y Cronjobs) con Microsoft SharePoint extraída vía MSAL. Se complementa con Web Scraping recursivo renderizando JS a través de Playwright, y búsquedas en tiempo real en DuckDuckGo.

  • Sincronización Delta Incremental
  • HTML a Texto Limpio (Deep Scraping)
  • Extracción PaddleOCR para PDFs Escaneados

Consulta Jurídica Inteligente

Integración en fase de prototipado técnico (desarrollo futuro) vía Model Context Protocol (MCP) con la API nativa del BOE Español. La arquitectura está sentada para buscar normativas consolidadas y resumir legislación de forma experimental.

  • Agente MCP FastMCP Nativo
  • Resolución Semántica (e.g. "Ley de Datos" -> BOE-A-2018-16673)
  • Summaries ejecutivos con Llama 3

BOE Explorer

Monitorización Avanzada

Control absoluto sobre los recursos del sistema gracias a la integración nativa con exportadores de NVIDIA y servidores de métricas. El sistema es transparente y auditable mediante gráficas enriquecidas.

  • Dashboards Visuales en Grafana
  • Telemetría y Latencias en Prometheus
  • Control de carga GPU y VRAM (DCGM Exporter)
